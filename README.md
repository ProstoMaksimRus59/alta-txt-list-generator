# alta-txt-list-generator
Для создание с алты двух файлов с топом для оправки их через другое(например в дс через бота)

Функции которые щас доступны

>ALTATEXTGENERAGOR.F1() - обновляет дб

>ALTATEXTGENERAGOR.toppla() - обновляет файл toppla.txt

>ALTATEXTGENERAGOR.topver() - обновляет файл toplvl.txt
